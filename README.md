# Honeycomb Oasis 3X Website

Official flat-file website for the Honeycomb Oasis 3X Rust server.

The website includes live Raid Weekend and monthly wipe countdowns, complete server rules, Purge Week information, Hive Events, Zorp Offline Protection details, and the Discord community link.

## Schedule

- Map wipe: last day of every month at 2:00 PM Eastern Time
- Raid Weekend: Friday 7:30 AM Eastern Time to Monday 7:30 AM Eastern Time
- Purge Week: final seven days before wipe, with raiding active 24/7
- Zorp Offline Protection: players remain protected while their protection is active

## PvP

- Main map: PvE
- PvP zones: all Monuments and the Hive Arena

## Hive Events

- Sting or Be Stung
- Raid Bases
- Docked Cargo
- Bradley Events
- Arena Battles
- Treasure Hunts
- Community Challenges

All website files are stored directly in the repository root for GitHub Pages.
